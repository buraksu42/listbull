# listbull — brand assets

Direction B (revised). Bull head silhouette in teal (#00D9C0) with two cobalt (#3D7DFF) check-marks as horns.

## Files

### svg/
- `listbull-mark.svg` — primary color mark, 1024×1024
- `listbull-mark-mono.svg` — monochrome (uses `currentColor` — drop into HTML and set `color`)
- `listbull-app-icon.svg` — full app icon tile with dark squircle background
- `listbull-lockup-light.svg` — horizontal mark + "listbull" wordmark, light backgrounds
- `listbull-lockup-dark.svg` — same lockup, white text, dark background baked in

### png/
Pre-rendered raster exports:
- `listbull-mark-{32,64,128,256,512,1024}.png` — favicon + general use
- `listbull-app-icon-{180,192,256,512,1024}.png` — iOS (180), Android (192), stores (1024)
- `listbull-lockup-light-{320,640,1280}.png`
- `listbull-lockup-dark-{320,640,1280}.png`

## Colors
- Teal `#00D9C0` — primary (bull head)
- Cobalt `#3D7DFF` — secondary (check-mark horns)
- Ink `#0A1419` — wordmark on light, background on dark

## Wordmark
"listbull" — Inter 700, letter-spacing −0.035em. No accent dot.
